Disable some Shipping Methods depending on Shipping Class

== Description ==

A simple tool to exclude Shipping Methods depending on the Shipping Class of products in the cart.

= How to use =

1. Go to Woocommerce -> Settings -> Shipping
2. Go to the subsection "    Shipping Methods by Classes".

There has many lists has you have defined Shipping Classes.

For each Shipping Class, check which Shipping Method has to be disabled.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
